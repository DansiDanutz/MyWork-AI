## Getting Started

1. **Install MyWork-AI**: `curl -fsSL https://mywork-ai.dev/install.sh | bash`
2. **Choose your use case**: Pick from the examples above
3. **Run the command**: Execute the template command
4. **Customize**: Modify generated code to fit your needs
5. **Deploy**: Use built-in deployment scripts
