# MyWork Framework Requirements

Date: 2026-01-26

## Must-Have (Phase 8)

- Unified payments + credits ledger for all order types
- Escrow automation (hold/release + audit trail)
- Reliable refunds and reversals across Stripe and credits
- Health checks + configuration validation in production

## Must-Have (Phase 9)

- Brain ingestion pipeline with embeddings and dedupe
- Quality scoring + provenance for Brain entries
- Search relevance and ranking improvements

## Must-Have (Phase 10)

- Audit review UX (seller-facing + admin view)
- Trust badges + listing verification flow
- CI-based audit integration

## Out of Scope (for now)

- White-label marketplace
- Multi-tenant org billing
- Legal automation workflows
