**PRODUCTION VERIFICATION CHECKLIST**

Open your Vercel deployment URL in a new browser window (incognito recommended for clean test).

### 1. Authentication Flow (2 min)

- [ ] Homepage loads without errors
- [ ] Click "Sign in with GitHub" button
- [ ] GitHub OAuth flow completes
- [ ] Redirected to /welcome page
- [ ] User avatar and name displayed correctly

### 2. Task Management (3 min)

- [ ] Navigate to /tasks (or dashboard)
- [ ] Create new task with title and description
- [ ] Task appears in task list
- [ ] Edit task (change title, description, status)
- [ ] Changes persist after page refresh
- [ ] Delete task - confirm dialog appears
- [ ] Task is removed from list

### 3. Task Organization (2 min)

- [ ] Add tags to a task
- [ ] Search for task by title
- [ ] Search results appear correctly
- [ ] Filter by status (todo, in-progress, done)
- [ ] Clear filters - all tasks visible

### 4. File Attachments (2 min)

- [ ] Edit a task
- [ ] Upload a small image file (<5MB)
- [ ] Thumbnail preview appears
- [ ] Upload a larger file (if TUS works)
- [ ] Download attached file - opens correctly
- [ ] Delete attachment

### 5. Feedback Widget (1 min)

- [ ] Click floating feedback button (bottom-right)
- [ ] Form opens
- [ ] Select feedback type
- [ ] Enter feedback text
- [ ] Submit - success message appears
- [ ] Widget closes

### 6. Performance Check (1 min)

- [ ] Open Chrome DevTools -> Lighthouse
- [ ] Run performance audit
- [ ] Performance score > 80
- [ ] LCP < 2.5s
- [ ] CLS < 0.1

### 7. Mobile Responsiveness (1 min)

- [ ] Open in mobile view (DevTools device toolbar)
- [ ] Navigation works (hamburger menu)
- [ ] Task list is scrollable
- [ ] Swipe gestures work (if on touch device)

### 8. Health Check (30 sec)

- [ ] Visit /api/health
- [ ] Response shows "status": "healthy"
- [ ] Database check shows "healthy"

**Expected Result:**
All checkboxes should be marked. If any fail, note the specific issue.
